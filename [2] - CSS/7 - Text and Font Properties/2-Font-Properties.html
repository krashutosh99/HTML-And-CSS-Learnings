[ 5. ] "font-size" property

Note: If you do not specify a font size, the default size for normal text, like paragraphs, is 16px (16px = 1em).

This is heading 1 with font-size = 60px;

This is a paragraph with font-size = 30px;

This is another paragraph with font-size = 20px;

This ia a paragraph with font-size = 2em [16px * 2 = 32px | 32px/16 = 2em ] .

[ 6. ] Shorthand - font : style, variant, weight, size/line-height, family.

This is a paragraph. The font is set to italic, small-caps and bold, the font size is set to 50 pixels, the line height is set to 40 pixels, and the font family is Georgia.
