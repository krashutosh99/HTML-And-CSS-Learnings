[4.] The alt Attribute

The required alt attribute for the image "img" tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection, or an error in the src attribute, or if the user uses a screen reader.

[5.] The style Attribute

The style attribute is used to add styles to an element, such as color, font, size, and more.
